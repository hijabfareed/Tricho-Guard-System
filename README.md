# Tricho Guard System

Tricho Guard is a modern AI-IoT healthcare dashboard for monitoring scalp health, reviewing AI-assisted hair disease predictions, and tracking real-time patient sensor data.

## Features

- Responsive React + Vite frontend
- Dark blue healthcare dashboard UI
- Mock authentication flow with protected routes
- Patient management and scalp analysis views
- IoT monitoring for temperature, moisture, and pH sensors
- AI hair disease prediction summaries
- Reports and profile management pages

## Development

```bash
npm install
npm run dev
```

## Validation

```bash
npm run lint
npm run build
```
